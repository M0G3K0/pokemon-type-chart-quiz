## Design Tokens

`pt-card`は以下のデザイントークンを使用します。

### Size

| Token | Padding | 用途 |
|-------|---------|------|
| `card.padding.sm` | 12px | コンパクトなカード |
| `card.padding.md` | 16px | 標準（デフォルト） |
| `card.padding.lg` | 24px | ゆったりしたカード |

### Elevation

| Token | Shadow | 用途 |
|-------|--------|------|
| `card.elevation.flat` | none | 境界線のみ |
| `card.elevation.raised` | small | やや浮いた印象 |
| `card.elevation.overlay` | large | 明らかに浮いた印象 |

### Layout Specifications

| Section | Role |
|---------|------|
| `pt-card-header` | タイトル、メタ情報、アイコン |
| `pt-card-content` | メインコンテンツ |
| `pt-card-footer` | アクションボタン、補足情報 |
