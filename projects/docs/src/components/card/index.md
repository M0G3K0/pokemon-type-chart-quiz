> 関連するコンテンツをグループ化するコンテナコンポーネント

## Overview

`pt-card`は、関連するコンテンツをグループ化して表示するためのMoleculeレベルのコンポーネントです。ヘッダー、コンテンツ、フッターの構造を持ち、一貫したスタイリングを提供します。

**複合コンポーネント**:
- `pt-card`: カード全体のコンテナ
- `pt-card-header`: ヘッダーエリア（タイトル、アクション）
- `pt-card-content`: メインコンテンツエリア
- `pt-card-footer`: フッターエリア（アクションボタン等）

## When to use ✅

| シチュエーション | 説明 |
|------------------|------|
| **関連情報のグループ化** | プロフィール、商品情報など |
| **アクション可能なコンテンツ** | フッターにボタンを配置 |
| **視覚的な区切り** | 他のコンテンツと明確に分離 |
| **階層構造の表現** | 浮いた印象で重要度を示す |

## When not to use ❌ (Anti-patterns)

| アンチパターン | 理由 | 代替案 |
|----------------|------|--------|
| **単純な背景ボックス** | Cardは構造を持つ | `pt-surface` |
| **カード内にカード** | 複雑さが増す | フラットな構造に再設計 |
| **1つのみの内部要素** | Cardの構造が不要 | `pt-surface` |

## Accessibility

- カード自体にはARIA属性不要
- 内部のインタラクティブ要素がアクセシブルであること
- フォーカス順序: ヘッダー → コンテンツ → フッター

## Related Components

- `pt-surface`: 単純なコンテナスタイリング
- `pt-button`: カードフッターのアクションボタン
- `pt-heading`: カードヘッダーのタイトル
