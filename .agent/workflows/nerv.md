# /nerv — NERV 研究ログ記録

ユーザーが `/nerv` と言ったら、直近の会話で得られた知見を `MAGI/NERV-LOG.md` に追記する。

## 手順

1. **記録内容の確認**

ユーザーに聞く:
- 何を記録したいか（知見 / 意思決定 / 検証結果 / TODO）
- 該当 Phase（0 / 1 / 2 / 3）

ユーザーが具体的に指示した場合はそのまま使う。
指示がなければ、直近の会話から重要な知見を自分で判断してまとめる。

2. **NERV-LOG.md への追記**

`MAGI/NERV-LOG.md` を開き、該当 Phase セクションの末尾に追記する。

追記フォーマット:
```markdown
### [タイトル] (YYYY-MM-DD)
- [知見の内容]
- **決定**: [意思決定があれば]
- **TODO**: [次のアクションがあれば]
```

3. **コミット（任意）**

ユーザーが望めばコミットする。コミットメッセージ:
```
docs(nerv): log [簡潔な内容] to NERV-LOG
```

ブランチは**現在のブランチ**（通常 `beta/nerv-phase1`）にコミット。

## 注意
- NERV-LOG.md が存在しない場合は作成する
- 既存の内容を上書きしない。**常に追記**
- 会話の知見がなくても、ユーザーが指定した内容をそのまま記録する
